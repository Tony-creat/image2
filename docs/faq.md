# 常见问题

## 余额不足时会提示什么？

工具会提示：

```text
余额不足，请充值后再试
```

## 请求失败时会提示什么？

上游服务不可用时会提示：

```text
请求失败，服务暂时不可用，请稍后重试
```

网络异常时会提示：

```text
网络连接失败，请检查网络或代理后重试
```

## 为什么输出目录没有响应 JSON？

响应 JSON 对普通使用者没有价值，还可能暴露请求细节。当前版本只在输出目录保存 PNG。本地诊断日志仍保留在程序目录中，便于开发者排查。

## EXE 可以复制到别的电脑运行吗？

可以。新电脑需要满足：

1. 能访问 ChunxueAPI。
2. 有可用的 API Key。
3. 输出目录有写入权限。

## 从源码运行需要哪些依赖？

主要第三方依赖是 `requests`。打包 EXE 需要 `pyinstaller`。
